Summary

This is an interactive Python pattern displaying program, created as my first programming project at university.

The task

The task for this project was to generate a predefined pattern based on the scale and colours defined by the user as a demonstartion of skill for my introduction to programing unit in my fisrt year of university.

Goals

  • The program should begin by prompting the user using a text-based interface to enter the patchwork size the desired three colours
  • The program must ensure are all the colours are different.
  • The programâ€™s user interface should be easy to use, friendly and robust.
  • On entering invalid data, the user should be re-prompted until the entered data is valid
  • A patchwork sample is made up of patches of two different designs and three colours.
  • Patchworks are square and can be of three different sizes: 5 x 5, 7 x 7 and 9 x 9.
  • The program must draw the patches using the facilities provided in the graphics module (Line, Circle etc.),
  • The program must not use bitmapped images

Stretch goals

  • After the patchwork design has been drawn, the program must allow the user to change the colour of patches by clicking on them with the mouse.
  • The colours must cycle in the order they were entered by the user
  • The user must be able to make as many colour changes to as many patches as they wish

Results

Each goal was met including the stretch goals resulting in an overall grade of 98% for this project upon presentation.

Reflection

This project was very enjoyable, in particular I enjoyed going back over the project once I had a working framework and finding ways to improve the robustness of the user interface
